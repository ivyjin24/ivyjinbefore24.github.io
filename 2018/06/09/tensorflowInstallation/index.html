tensorflowInstallation

安装步骤

1.安装docker

docker安装地址：https://www.docker.com/get-docker
注册的时候可能会需要翻墙

2.启动docker

mac右上角点docker图标启动

3.用docker安装jpyter notebook

首次运行会安装tensorflow image服务，速度可能会比较慢;最后打开http://localhost:8888就可以使用jupyter notebook

4.停止/启动容器

停止：control+c 停止容器
启动：
docker ps -a 查看容器ID;
docker start -i (跟上面查到的容器ID)

5.jupyter读取（挂载）本地文件

6.异常处理

可能会提示端口被占用，是因为挂载需要在CONTAINER创建时挂载；
这时候需要先执行

查看所有CONTAINER，然后删除所有CONTAINER

7.jupyter执行终端命令

在命令前加!即可，如!ls
