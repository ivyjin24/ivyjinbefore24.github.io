SQL时间函数

1.日期与unix时间戳转换

1.1 获取当前unix时间戳：

unix_timestamp()

1.2 unix时间戳转日期

from_unixtime(bigint unixtime[, string format])
示例：from_unixtime(1529931158,’yyyy-MM-dd’)

1.3 日期转unix时间戳

unix_timestamp(string date)
示例：unix_timestamp(‘2018-06-25 12:00:00’)

1.4 指定日期格式转unix时间戳

unix_timestamp(string date, string pattern)
示例：unix_timestamp(‘20180625’,’yyyyMMdd’)

2.计算时间差

2.1 计算相差的天数

datediff(string enddate, string startdate)

2.2 计算相差的秒数

将时间转化为unix时间戳后，计算时间戳的差，得到的值即为秒数
unix_timestamp(‘2017-04-13 23:59:59’) - unix_timestamp(‘2017-04-13 22:59:59’)

3.日期增加/减少

3.1 日期增加

date_add(string startdate, int days)

3.2 日期减少

date_sub (string startdate, int days)
