一元
ceil：大于等于该值的最小整数

二元
subtract：从第一个数组中，减去第二个数组中的元素
power：将第一个数组中的元素A，根据第二个数组中的相应元素B，计算指数
copysign：将第二个数组中值的符号复制给第一个数组中的值
mod：元素级求模（除法的余数）

seed：确定随机数生成器的随机种子

SQL时间函数

1.日期与unix时间戳转换

1.1 获取当前unix时间戳：

unix_timestamp()

1.2 unix时间戳转日期

from_unixtime(bigint unixtime[, string format])
示例：from_unixtime(1529931158,’yyyy-MM-dd’)

1.3 日期转unix时间戳

unix_timestamp(string date)
示例：unix_timestamp(‘2018-06-25 12:00:00’)

1.4 指定日期格式转unix时间戳

unix_timestamp(string date, string pattern)
示例：unix_timestamp(‘20180625’,’yyyyMMdd’)

2.计算时间差

2.1 计算相差的天数

datediff(string enddate, string startdate)

2.2 计算相差的秒数

将时间转化为unix时间戳后，计算时间戳的差，得到的值即为秒数
unix_timestamp(‘2017-04-13 23:59:59’) - unix_timestamp(‘2017-04-13 22:59:59’)

3.日期增加/减少

3.1 日期增加

date_add(string startdate, int days)

3.2 日期减少

date_sub (string startdate, int days)

文件一需要做的处理：

position1_label：上一节点名称

position2_label：本节点名称

category：position1_label-position2_label

position2：
对concat(level1,coalesce(level2,level1),coalesce(level3,level2,level1))进行排序（其中coalesce函数表示，取第一个不为空的值），level3为对应顺序，level2为在根据level1,level2计算的排序的均值（可能有小数，通过floor函数取整数），level1则为根据level1计算的均值。

position1:
在tableau中计算，公式为floor({fixed position1_label:avg(position2)})如果position1_label有重复值可能会出现对应不上的情况，可以通过同时取上其他值构建一个唯一的position1_label，比如level1，level2等。
