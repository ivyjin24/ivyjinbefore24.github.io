发表于 2018-11-25 | 分类于 python

作为一个转专业的分析师，入行之前做图表基本都是Excel搞定，所以在学习用python实现可视化的时候，总还是习惯对标到Excel来理解。在Excel里面插入图表，第一步是选择图表类型；然后添加表元素，表元素包含了图表标题、坐标轴标题、坐标轴、数据标签、网格线和图例；同时可以对修改颜色、移动表的位置等等……对于matplotlib来说，基本也是具备这些功能的，目前也计划根据这个逻辑逐步梳理出来，方便之后的查询和代码复用。

SPSS Modeler连接spark

发表于 2018-10-16 | 分类于 SPSS Modeler

1.连接ODBC数据源

下载ODBC manager
打开后在系统DSN添加Simba Spark ODBC Driver
点击配置，增加HOST、PORT和DATABASE
其中HOST,DATABASE是字符串型，直接配置可能连不上,需要在 /Library/ODBC 的odbc.ini输入

2.在SPSS Modeler中配置

双击SQL -> SQL查询 -> 数据源（添加新数据库连接） -> 选择前面添加的数据源，录入用户名和密码 -> 连接
