利用python进行数据分析学习笔记

1.全书概览

《利用python进行数据分析》是用python做数据分析的一本入门读物，主要内容包括：环境搭建，数据加载，数据处理（numpy,pandas、数据聚合和分组运算），数据存储，数据可视化（matplotlib、pandas），和时间序列。

2.环境搭建

python可以在IDE、用电脑终端运行Ipython和jupyter Notebook…… jupyter Notebook对新手操作最友好
最方便的安装anaconda，在Environments中创建一个环境，然后就可以在Home里面launch一个jupyter Notebook了

4.数据处理

4.1.Numpy

4.1.1 数组创建函数
array：将输入数据转化为ndarray
asarray：将输入转化为dnarray，如果本身是一个ndarray就不进行复制
np.arrange：接reshape限制形状
ones、one_like：根据指定形状创建一个全是1的数据
zeros、zeros_like：根据指定形状创建一个全是零的数据
empty、empty_like：创建指定形状的空数组
eye、identity：创建一个正方形的N*N单位矩阵（对角线为1，其余为零）
4.1.2 ufunc
一元
ceil：大于等于该值的最小整数
floor：小于等于该值得最大整数
modf：将数组的整数和小数部分以两个独立的数组的形式返回
二元
subtract：从第一个数组中，减去第二个数组中的元素
power：将第一个数组中的元素A，根据第二个数组中的相应元素B，计算指数
copysign：将第二个数组中值的符号复制给第一个数组中的值
mod：元素级求模（除法的余数）
4.1.3 统计方法
sum\mean\std\var\min\max\argmin\argmax
cumsum(所有元素的累积和)\cumprod(所有元素的累计积)
numpy.linalg函数
dot：矩阵乘法
trace：对角线元素的和
inv：矩阵的逆
Pinv：矩阵的违逆
4.1.4 随机数生成：np.random
seed：确定随机数生成器的随机种子
randint\randn(正态分布，平均值为0，标准差为1)\binomial(二项式分布)\normal(正态（高斯）分布)

4.2 pandas

Series：定长的有序字典，在算术运算中会自动对齐不同索引的数据
DataFrame：
index方法
append\diff\intersection(交集)\union(并集)\insert\drop
约简方法的选项
axis：轴，0为行，1为列
skipna：排除缺失值
level：层次化索引的聚合维度
